FlexBox's

to start with working with FlexBox's you need to create a container that uses the display: flex property. in a CSS script

.flex-container {
    display: flex;
}

.flex-container > div {
   background-color: lightgray;
   margin: 10px;
   padding: 20px;
   font-size: 30px;
}

Introduction to pseudo classes:

with pseudo classes you can add logic to your css files!
the first thing we can do is use the :hover pseudo-class

   .flex-container > div:hover {
       background-color: gray;
        color: white;
   }

what this does is when your mouse is hovering over the flex container
it will run the command in the currly brakets,
in this case tuen the color of the box gray and the color of the text white
